# Cloud-Enabled Smart Home Automation

A full-stack web application that allows users to remotely control and monitor virtual smart devices through a centralized dashboard with cloud-connected services.

## 🚀 Features

* User authentication (Login / Register)
* Smart device controls:

  * Light ON/OFF
  * Fan ON/OFF
  * Door Open/Close
  * AC ON/OFF
* Voice command support
* Real-time dashboard updates
* Activity logs and monitoring
* Responsive UI for mobile and desktop
* Export activity reports to AWS S3
* Download CSV reports

## ☁️ Cloud Integration

* Frontend Hosting: Vercel
* Backend Hosting: Render
* Database: MongoDB Atlas
* Cloud Storage: AWS S3

### Cloud / Deployment

* AWS S3

## 📄 AWS S3 Report Export

The application generates smart home activity reports in CSV format and uploads them to AWS S3 for secure cloud storage and download access.
